# DIY ESP32 Snake Game Console 🎮

A tiny handheld Snake gaming console made using an ESP32, SSD1306 OLED display and joystick module.

This project transforms a simple ESP32 into a mini retro gaming device with smooth controls and classic Snake gameplay.

Perfect for DIY electronics enthusiasts, retro gaming fans and ESP32 beginners.
For the full step-by-step hardware build guide and component recommendations, check out the [ESP32 OLED Snake Game Instructable](https://www.instructables.com/-ESP32-OLED-Snake-Game-Best-SSD1306-and-Joystick-S/).

## 🧰 Components Required

- ESP32 Development Board
- SSD1306 OLED Display (128x64 I2C)
- Joystick Module
- Breadboard
- Jumper Wires

---

## 🔌 Wiring

### OLED Display

| OLED | ESP32 |
|---|---|
| VCC | 3.3V |
| GND | GND |
| SDA | GPIO 21 |
| SCL | GPIO 22 |

---

### Joystick Module

| Joystick | ESP32 |
|---|---|
| VRX | GPIO 34 |
| VRY | GPIO 35 |
| SW | GPIO 27 |
| VCC | 3.3V |
| GND | GND |

---

## 📚 Libraries Used

Install these libraries in Arduino IDE:

- U8g2
- Wire

---

## 🚀 Uploading the Code

1. Open the `.ino` file in Arduino IDE
2. Select your ESP32 board
3. Install required libraries
4. Connect the ESP32
5. Upload the code
6. Enjoy your DIY handheld Snake console 🎮
